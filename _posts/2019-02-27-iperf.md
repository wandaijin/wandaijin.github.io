`iperf`命令用来测试TCP，UDP，STCP网速。使用是通过服务器端和客户端进行数据传输来测试网速。

>-u UDP模式

>-P 连接数

>-p 端口号默认5001，服务端和客户端需要一致

>-t 测试时间(秒)

>-f [bkmaBKMA] 带宽输出单位

- 服务端开启命令
```bash
$ iperf -s
```
- 客户端连接命令
```bash
$ iperf -c ip
```
