- 典型用法
```bash
$ dig @8.8.8.8 www.wandaijin.com # dig @DnsServerOrDnsIP DomainName type
```
_如果dig后面没有参数，将直接查询根域名(".")信息，如果没有指明DNS服务器则尝试依次使用/etc/resolv.conf中的服务器配置信息_

- DNS反向查找
```bash
$ dig -x 8.8.8.8
```
